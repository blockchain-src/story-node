# 步骤
---
## 1️⃣切换到root用户
```
sudo -i
```
---
## 1️⃣克隆 github 仓库
```
git clone https://github.com/blockchain-src/story-node.git && cd story-node
```
---
## 2️⃣运行一键脚本：安装缺失的依赖项和运行节点
```
chmod +x story.sh && ./story.sh
```
一键脚本自动安装缺失的软件包、安装story节点、配置钱包、设置验证器等。
